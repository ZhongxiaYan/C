Projects
  • WICED Mouse
    My partner and I utilized the sensors in the WICED Sense tag to create a hover mouse. I modified the existing Android app base to communicate with a Java computer app via UDP packets. A magnet is attached to the users thumb, and we interpreted tilt data for mouse movement and magnetometer data for clicks and scrolls.
  • Telemetry Grapher
    Using JFreeChart, I created a real time graphical interface for the Cal Solar Vehicle team's telemetry program. The program plotted received power and speed, and displayed voltages and temperatures in tables. The GUI can also take existing log datas and simulate the car's messages by timestamps recorded by the log.
  • Paint
    Created Paint as a Chrome Extension with some customized useful features, such as a flexible arc tool, typing Greek characters using LaTeX syntax. In addition, basic features like flood fill, rectangle, redo and undo, and upload or download image are also available.
  • Dashboard Design
    I redesigned the Dashboard circuit board schematics and layout for Cal Solar Vehicle Team, basing some of the designs on a previous revision. The board communicates with an array of I2C and SPI devices, including the accelerator and brake pedal, 7-segment LED display, and various switches and LED drivers.
